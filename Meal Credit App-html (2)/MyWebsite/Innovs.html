Arduino Uno R3

The Arduino Uno is a microcontroller, meaning it's a small computer doing basics tasks often through circuitry. Code for it is a modified C/C++. Main components include: ultrasonic sensors, LEDs, resistors, transistors, capacitors, servo motors, and stepper motors. Shields are very common for WiFi, Bluetooth, and Radio transmission signaling/data transfer.

App Inventor Companion

App Inventor's Companion App allows the immediate transfer of data between electronic devices through WiFi connection. It will noticably automatically refresh after a Designer change and unnoticably with a Block change. Android only unfortunately. Requires input of character code or QR code for connection.
